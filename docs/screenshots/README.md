# Ekran Görüntüleri

Ana README bu klasördeki 4 görsele referans verir. Dosyaları buraya ekleyin:

| Dosya | İçerik | Sayfa |
|-------|--------|-------|
| `landing.png` | Sinematik giriş (NOVA başlığı + CTA) | `/` |
| `eda.png` | Otomatik EDA dashboard'u | `/studio/eda/[id]` |
| `train.png` | Model eğitimi (gerçek adımlı SSE günlüğü + metrikler) | `/studio/train/[id]` |
| `predict.png` | Tahmin (dramatik reveal + güven) | `/studio/predict/[id]` |

## Nasıl çekilir

Backend + frontend'i çalıştırın, bir CSV yükleyip modeli eğitin, ardından her
sayfada tam ekran görüntüsü alıp `docs/screenshots/` altına yukarıdaki adlarla
kaydedin (önerilen genişlik ~1440px, karanlık tema).
